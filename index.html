Política de Privacidad

Última actualización: 1 de marzo de 2026

1. Información que recopilamos

Bounce puede recopilar los siguientes datos cuando el usuario decide crear un perfil o iniciar sesión con Google:

  • Nombre de usuario elegido por el jugador (permanente una vez establecido)
  • Foto de perfil de la cuenta de Google, si el usuario inicia sesión con Google
  • Dirección de correo electrónico, únicamente si el usuario inicia sesión con Google
  • Puntuaciones y récords para los rankings globales
  • Progreso de desafíos diarios y racha de días consecutivos

El uso de cuenta es completamente opcional. La aplicación puede usarse de forma anónima sin proporcionar ningún dato personal.

2. Datos almacenados localmente

Con independencia de si el usuario tiene cuenta, la app guarda en el dispositivo:

  • Puntuación máxima personal e historial de partidas
  • Monedas y mejoras adquiridas en la tienda
  • Preferencias de idioma y ajustes
  • Cola de sincronización offline (puntuaciones pendientes de subir)

3. Datos almacenados en la nube

Si el usuario crea un perfil o inicia sesión con Google, los siguientes datos se almacenan en Firebase Firestore (Google Cloud):

  • Nombre de usuario y avatar seleccionado
  • Foto de perfil de Google (URL)
  • Puntuaciones para los rankings diario, semanal, mensual y global
  • Fecha de última partida y mejor puntuación

Estos datos son necesarios para el funcionamiento de los rankings en tiempo real y la sincronización entre dispositivos.

4. Uso offline y sincronización

Bounce funciona completamente sin conexión a internet. Las partidas jugadas sin conexión se guardan localmente y se sincronizan con los rankings en la nube de forma automática en cuanto el dispositivo recupera la conexión.

5. Permisos

Bounce solicita los siguientes permisos en Android:

  • VIBRATE — para feedback táctil durante el juego
  • INTERNET — para sincronización con Firebase y login con Google

No se solicita acceso a cámara, micrófono, contactos, ubicación ni almacenamiento externo.

6. Servicios de terceros

Bounce utiliza los siguientes servicios de Google:

  • Firebase Authentication — para el inicio de sesión con Google y gestión de sesiones anónimas
  • Firebase Firestore — base de datos en la nube para perfiles y rankings
  • Firebase Hosting — para servir la aplicación web

Estos servicios están sujetos a la Política de Privacidad de Google. No se integra ninguna red publicitaria, servicio de analítica ni herramienta de seguimiento de terceros.

7. Retención y eliminación de datos

El usuario puede eliminar su cuenta y todos sus datos asociados en cualquier momento cerrando sesión desde la pantalla de perfil. Para solicitar la eliminación completa de los datos almacenados en Firebase, el usuario puede contactar con el desarrollador en la dirección indicada a continuación.

8. Menores de edad

Bounce es apto para todas las edades. No recopilamos datos personales de forma intencionada de menores de 13 años. El uso de cuenta con Google requiere que el usuario cumpla los requisitos de edad de Google.
